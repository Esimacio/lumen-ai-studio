# Support

Lumen AI Studio is an experimental community fork provided as-is.

I am not the original developer and cannot guarantee fixes for device-specific crashes, model compatibility issues, build errors, operating-system problems, or local setup issues.

Bug reports and contributions are welcome, but support is best-effort only.

## Before opening an issue

Please include:

- Operating system and version
- App version or commit used
- Hardware information, if relevant
- Model file used, if relevant
- Steps to reproduce the problem
- Error message, logs, or screenshot

Issues without enough detail may be closed.
