# Contributing

Thanks for your interest in Lumen AI Studio.

This project is an experimental community fork focused on creating a calm, practical local AI studio for open-source tools.

## Project direction

Lumen AI Studio aims to be:

- Local-first
- Open-source friendly
- Practical and calm
- Useful for working with AI tools on your own device

It is not intended to be positioned as an “uncensored,” jailbreak, bypass, or illegal-content-focused project.

## Ways to contribute

Helpful contributions include:

- Documentation improvements
- Bug reports with clear reproduction steps
- UI and wording cleanup
- Build and packaging improvements
- Platform testing
- Accessibility improvements

## Pull requests

Please keep pull requests focused and small when possible.

Good examples:

- Fix one broken instruction
- Improve one UI label
- Update one setup step
- Clean one part of the README

Avoid mixing large technical changes with branding or documentation changes in the same pull request.
